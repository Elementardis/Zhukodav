# Custom UI assets

Put PNG files here with the exact names below. Missing files are okay: the game will use the existing Pixi-drawn UI.

Recommended source sizes:

- `level_entry_icon.png` - `192x192`
  Icon shown in the level entry popup.
- `level_tile_completed.png` - `128x128`
  Completed level tile background.
- `level_tile_locked.png` - `128x128`
  Not-yet-completed level tile background.
- `game_hud_panel.png` - `1180x112`
  Top in-level HUD panel.
- `game_goal_badge.png` - `220x72`
  Goal counter badge inside the HUD.
- `game_playfield_frame.png` - `900x520`
  Main in-level playfield frame.
- `game_side_panel.png` - `128x360`
  Left/right color-button side panel.
- `game_settings_button.png` - `96x96`
  Settings button background.

Keep transparent padding inside the PNG if the art needs glow, shadow, or soft edges.
